# student_prediction
# Student Career Recommendation using ANN

## Overview
This project implements an Artificial Neural Network (ANN) to recommend suitable career paths for students based on their academic performance and skill-related features.

The system analyzes student data and predicts careers such as Engineer, Doctor, Data Scientist, etc.



## Objectives
- Build an ANN model for career prediction  
- Analyze student academic and skill-based features  
- Recommend suitable career paths  
- Evaluate model performance using:
  - Accuracy  
  - Precision  
  - Recall  
  - F1-score  



## Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- TensorFlow / Keras  
- Matplotlib  



## Project Workflow
1. Load dataset  
2. Data preprocessing (handle missing values)  
3. Feature selection  
4. Label encoding  
4. Data normalization  
6. Train-test split  
7. Build ANN model  
8. Train model  
9. Evaluate performance  
10. Predict career  



## ANN Model Architecture
- Input Layer → Student features  
- Hidden Layer 1 → 64 neurons (ReLU)  
- Hidden Layer 2 → 32 neurons (ReLU)  
- Output Layer → Softmax (Career classes)  



## Evaluation Metrics
- **Accuracy** → Overall correctness  
- **Precision** → Correct positive predictions  
- **Recall** → Coverage of actual positives  
- **F1-score** → Balance between precision & recall  



## Results
- High accuracy in career prediction  
- Balanced precision and recall  
- Reliable recommendations for new students  



## Conclusion
This project demonstrates how Artificial Neural Networks can be effectively used to guide students in choosing suitable career paths. The system provides accurate and scalable career recommendations using machine learning techniques.
